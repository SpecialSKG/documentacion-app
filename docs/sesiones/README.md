# Cierres de Sesión

Carpeta para handoffs de trabajo por sesión.

## Convención de archivo
Usa este formato:

`cierre_sesion_YYYY-MM-DD_HH-mm.md`

Ejemplo:

`cierre_sesion_2026-03-17_09-40.md`

## Flujo recomendado
1. Al terminar, dile a Codex: `cerremos sesión`.
2. Codex toma la plantilla `PLANTILLA_CIERRE_SESION.md`.
3. Codex crea un nuevo archivo de cierre en esta carpeta con fecha y hora.
4. El siguiente día, abre el último cierre y el contexto consolidado para retomar rápido.

## Nota
Para este proyecto, la ubicación recomendada es `docs/sesiones/` (no la raíz), porque mantiene historial y contexto en un solo lugar.
